# Maqsudur Rahman Academic Website

Static GitHub Pages website for https://mrrajon04.github.io

## Deployment
1. Create a GitHub repository named `mrrajon04.github.io`.
2. Upload all files from this folder to the repository root.
3. Go to Settings > Pages and select `Deploy from a branch`, branch `main`, folder `/root`.
4. Your site will be available at https://mrrajon04.github.io after GitHub Pages finishes deployment.
