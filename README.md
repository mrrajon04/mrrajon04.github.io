# Maqsudur Rahman Academic Website

Repository name: `mrrajon04.github.io`

## Upload instructions
1. Upload the contents of this folder directly to the root of the GitHub repository.
2. Ensure that `index.html`, `styles.css`, `publications.html`, `supervision.html`, and the `assets/` folder are visible in the repository root.
3. In GitHub, go to Settings -> Pages and select Deploy from a branch -> main -> /root.
4. The website will be available at https://mrrajon04.github.io

## Editing updates
To add a new update, open `index.html`, search for `<section id="updates">`, and add a new `<li>` at the top of the ordered list.

## Replacing CV or documents
Replace the PDF files inside `assets/docs/` with updated versions using the same filenames.

## Replacing photo
Replace `assets/profile.jpg` with a new professional headshot using the same filename.
